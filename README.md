# Python Chat Application 💬

This project is a real-time Chat Application built using Python and Socket Programming.

## 🔹 Features
- Real-time message communication
- Client-Server architecture
- Multiple users can connect
- Simple and user-friendly interface

## 🔹 Technologies Used
- Python
- socket module
- threading module

## 🔹 How It Works
- The server waits for client connections.
- Clients connect to the server using IP address and port.
- Messages are sent and received instantly between connected users.

## 🔹 Learning Outcome
- Understanding of networking concepts
- Hands-on experience with socket programming
- Knowledge of multi-threading
